A widget that displays a list of scheduled posts in your sidebar.

== Description ==

This widget displays a list of scheduled posts in your sidebar. 

*   The number of scheduled posts displayed can be configured
*	You can configure the widget to only list scheduled posts from a specific category.

== Installation ==

1. Upload `subzane-upcoming-posts-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to your sidebar
